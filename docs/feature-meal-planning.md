# Feature: Meal Planning com Shopping List

Receitas isoladas nao resolvem o problema real do dia-a-dia: planejar o que comer na semana e saber o que comprar. Esta feature transforma o RecipeBox de um catalogo passivo em uma ferramenta de planejamento alimentar.

O usuario monta o cardapio da semana numa grade visual (7 dias x refeicoes), arrastando receitas para os slots. A partir do plano, gera automaticamente uma lista de compras com ingredientes consolidados e agrupados — pronta para levar ao mercado.

## Resultado Esperado

1. Criar um plano semanal associando receitas a dias e refeicoes
2. Visualizar o plano em formato de grade (7 dias x tipos de refeicao)
3. Gerar uma lista de compras consolidada a partir do plano
4. Marcar itens da lista como comprados/ja disponiveis
5. Navegar do plano diretamente para a receita completa
