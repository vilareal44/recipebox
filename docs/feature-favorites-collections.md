# Feature: Favoritos, Colecoes e Tags

Hoje o RecipeBox organiza receitas apenas por categoria fixa. Mas a forma como as pessoas pensam sobre receitas e mais rica: uma mesma receita pode ser "favorita", "sem gluten", "da vovo" e "pra quando vem visita" ao mesmo tempo. Esta feature permite organizacao pessoal e livre.

Tres mecanismos complementares: **favoritos** (um clique para marcar as receitas do dia-a-dia), **colecoes** (agrupamentos livres como "playlists" — ex: "Receitas de Natal", "Rapidas < 30min") e **tags** (etiquetas livres como `vegano`, `comfort-food`, `marmita` com autocomplete e filtro combinado).

## Resultado Esperado

1. Favoritar/desfavoritar qualquer receita com um clique (coracao no card e na pagina de detalhe)
2. Ver todos os favoritos em uma pagina dedicada
3. Criar colecoes com nome, descricao e cor
4. Adicionar e remover receitas de colecoes
5. Navegar por colecoes a partir do menu principal
6. Adicionar tags livres a qualquer receita (com autocomplete de tags existentes)
7. Filtrar receitas por combinacao de tags
8. Ver todas as tags em uso e quantas receitas cada uma tem
